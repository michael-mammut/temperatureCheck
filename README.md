

# Temperature alert
Check the temperature of my aquarium and send me a push notification

### Device
* Raspberry Pi 3
* Python

### APIs
* w1thermsensor (https://github.com/timofurrer/w1thermsensor)
* pushbullet (https://pypi.org/project/pushbullet.py/0.9.1/)
